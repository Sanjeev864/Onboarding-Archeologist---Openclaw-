# API Reference

Use the interactive OpenAPI docs at `http://localhost:8000/docs`.

Key endpoints:

- `POST /api/analyze`
- `POST /api/analyze/advanced`
- `GET /api/analysis/{repository_id}/scar-tissue`
- `GET /api/analysis/{repository_id}/bus-factor`
- `GET /api/analysis/{repository_id}/onboarding/{experience_level}/{role}`
- `POST /api/oracle/ask`
- `POST /api/webhooks/github/subscribe`
- `POST /api/webhooks/github/event`
- `GET /api/repositories/{repository_id}/export`
- `GET /api/metrics`
