# Contributing

Use focused branches and small pull requests.

## Standards

- Python: FastAPI, SQLAlchemy, typed service boundaries.
- TypeScript: strict mode, componentized UI, accessible controls.
- Tests: add focused tests for new analysis behavior and API endpoints.

## Workflow

1. Create a branch.
2. Implement with tests.
3. Run frontend build and backend tests.
4. Open a pull request with screenshots or API examples when useful.
